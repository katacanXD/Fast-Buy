## RU

FastBuy - мод, который мгновенно покупает бусты на PrisonEvo.

## Создатель

### Katacan

## Команды

#### `/fastbuy <кол-во покупок> <номер буста>` - покупает бусты

1 - Бустер монет

2 - Бустер шардов

3 - Бустер опыта ремесел
